🔧 GASバックエンドセットアップガイド

このガイドでは Google Apps Script（GAS） を使ったバックエンドを設定し、Green Carbon HRツールのデータを Google Sheets に保存・チーム共有できるようにする手順を説明します。所要時間: 約20〜30分。

📋 目次

  1. Google Sheetsの準備
  2. Apps Scriptにコードを貼り付け
  3. Web Appとして公開
  4. HRツールに接続
  5. トラブルシューティング

📊 Step 1: Google Sheetsの準備

1

スプレッドシートを作成

  1. Google Sheets を開き、新規スプレッドシートを作成
  2. ファイル名を Green Carbon HR Data に変更
  3. シート名を以下の通りに作成（タブを追加）：
    • 📋 Candidates — 採用候補者
    • 👥 Employees — 社員名簿
    • 📝 OneOnOne — 1on1記録
    • 🎯 Retention — 離職リスク
💡 スプレッドシートのURLにある長いIDをメモしておいてください。後で使います。
2

ヘッダー行を設定

CandidatesシートのA1から横に以下を入力：

id | name | email | position | status | date | source | note | createdAt | updatedAt

EmployeesシートのA1から：

id | name | role | dept | email | ext | start | location | manager | createdAt

⚙️ Step 2: Apps Scriptにコードを貼り付け

3

Apps Scriptを開く

  1. スプレッドシートのメニューから 拡張機能 → Apps Script を選択
  2. 既存のコードを全て削除
  3. 以下のコードを貼り付け：
const SHEET_ID = SpreadsheetApp.getActiveSpreadsheet().getId();
const ss = SpreadsheetApp.getActiveSpreadsheet();

function doGet(e) {
  const action = e.parameter.action;
  let result;

  try {
    switch(action) {
      case 'ping':
        result = {status: 'ok', message: 'Green Carbon GAS Backend Running'};
        break;
      case 'getCandidates':
        result = getCandidates();
        break;
      case 'getEmployees':
        result = getEmployees();
        break;
      default:
        result = {status: 'error', message: 'Unknown action: ' + action};
    }
  } catch(err) {
    result = {status: 'error', message: err.toString()};
  }

  return ContentService
    .createTextOutput(JSON.stringify(result))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const data = JSON.parse(e.postData.contents);
  const action = data.action;
  let result;

  try {
    switch(action) {
      case 'saveCandidate':
        result = saveCandidate(data.candidate);
        break;
      case 'saveEmployee':
        result = saveEmployee(data.employee);
        break;
      default:
        result = {status: 'error', message: 'Unknown action'};
    }
  } catch(err) {
    result = {status: 'error', message: err.toString()};
  }

  return ContentService
    .createTextOutput(JSON.stringify(result))
    .setMimeType(ContentService.MimeType.JSON);
}

function getCandidates() {
  const sheet = ss.getSheetByName('Candidates');
  const rows = sheet.getDataRange().getValues();
  const headers = rows[0];
  const candidates = rows.slice(1).map(row => {
    const obj = {};
    headers.forEach((h, i) => obj[h] = row[i]);
    return obj;
  }).filter(c => c.id);
  return {status: 'ok', candidates};
}

function saveCandidate(candidate) {
  const sheet = ss.getSheetByName('Candidates');
  const rows = sheet.getDataRange().getValues();
  const ids = rows.slice(1).map(r => r[0]);
  const now = new Date().toISOString();

  const idx = ids.indexOf(candidate.id);
  const rowData = [
    candidate.id, candidate.name, candidate.email,
    candidate.position, candidate.status, candidate.date,
    candidate.source, candidate.note,
    idx < 0 ? now : rows[idx+1][8],  // createdAt
    now  // updatedAt
  ];

  if (idx >= 0) {
    sheet.getRange(idx + 2, 1, 1, rowData.length).setValues([rowData]);
  } else {
    sheet.appendRow(rowData);
  }
  return {status: 'ok', message: 'Saved'};
}

function getEmployees() {
  const sheet = ss.getSheetByName('Employees');
  const rows = sheet.getDataRange().getValues();
  const headers = rows[0];
  const employees = rows.slice(1).map(row => {
    const obj = {};
    headers.forEach((h, i) => obj[h] = row[i]);
    return obj;
  }).filter(e => e.id);
  return {status: 'ok', employees};
}

function saveEmployee(employee) {
  const sheet = ss.getSheetByName('Employees');
  const rows = sheet.getDataRange().getValues();
  const ids = rows.slice(1).map(r => r[0]);
  const now = new Date().toISOString();

  const idx = ids.indexOf(employee.id);
  const rowData = [
    employee.id, employee.name, employee.role, employee.dept,
    employee.email, employee.ext, employee.start,
    employee.location, employee.manager, now
  ];

  if (idx >= 0) {
    sheet.getRange(idx + 2, 1, 1, rowData.length).setValues([rowData]);
  } else {
    sheet.appendRow(rowData);
  }
  return {status: 'ok', message: 'Saved'};
}

🚀 Step 3: Web Appとして公開

4

デプロイ

  1. 右上の 「デプロイ」→「新しいデプロイ」 をクリック
  2. 「種類を選択」で 「ウェブアプリ」 を選択
  3. 設定：
    • 説明: Green Carbon HR Backend v1
    • 次のユーザーとして実行: 自分（your email）
    • アクセスできるユーザー: 全員（または「Googleアカウントを持つ全員」）
  4. 「デプロイ」 をクリック
  5. 表示された ウェブアプリURL をコピー
⚠️ URLは https://script.google.com/macros/s/... で始まります。このURLは外部公開されるため、取り扱いに注意してください。
💡 コードを変更した場合は、「新しいデプロイ」ではなく「デプロイを管理」→「編集」→バージョンを「新バージョン」にして更新してください。

🔗 Step 4: HRツールに接続

5

採用管理バックエンド版に接続

  1. 採用管理バックエンド版 を開く
  2. 上部の入力欄にコピーしたURLを貼り付け
  3. 「接続テスト」 をクリック
  4. ✅ 接続済み と表示されれば完了
  5. 「🔄 同期」 でGoogle Sheetsのデータを取得
