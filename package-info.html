🚀 Vercelデプロイ 手順サマリー

1

GitHubリポジトリを作成

github.com/new でプライベートリポジトリを作成（名前例: gc-hr-tools）

2

このフォルダをプッシュ

cd gc_package && git init && git add . && git commit -m "Initial"
git remote add origin https://github.com/YOUR_ORG/gc-hr-tools.git
git push -u origin main
3

Vercelでインポート

vercel.com/new → 「Import Git Repository」→ gc-hr-tools を選択 → Deploy

4

URLをチームに共有

デプロイ完了後、https://gc-hr-tools-xxxx.vercel.app のURLをSlackなどで共有

5

（任意）GASバックエンドを接続

採用管理バックエンド版からGAS Web App URLを設定してチームデータを共有
