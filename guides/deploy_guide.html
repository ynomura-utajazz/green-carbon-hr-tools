🚀 Vercelデプロイガイド

Green Carbon HRツールパッケージを Vercel にデプロイして、チーム全員がブラウザからアクセスできるようにする手順です。GitHubアカウントがあれば無料で完了できます。

📋 デプロイ方法の選択

⭐ 方法A: GitHub + Vercel連携（推奨）おすすめ

GitHubにファイルをプッシュするだけで自動デプロイ。ファイルを変更するたびに自動で更新されます。

メリット: 自動デプロイ、バージョン管理、チームでの共同編集が容易

🖱️ 方法B: Vercel CLIでローカルデプロイ

コマンドラインから直接デプロイ。GitHubを使わずに素早くデプロイできます。

📁 方法C: ドラッグ&ドロップ（最も簡単）

Vercelのダッシュボードにフォルダをドラッグするだけ。最も手軽ですが、更新の都度手動操作が必要です。

🅰️ 方法A: GitHub + Vercel連携

1

GitHubリポジトリの作成

  1. GitHub で新規リポジトリを作成
  2. リポジトリ名: gc-hr-tools
  3. Visibility: Private（社内情報のため）
  4. 「Create repository」をクリック
2

ファイルをGitHubにアップロード

ターミナルで以下を実行（gc_packageフォルダ内から）：

cd /path/to/gc_package
git init
git add .
git commit -m "Initial commit: Green Carbon HR Tools"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/gc-hr-tools.git
git push -u origin main
💡 GitHubのリポジトリ画面から「uploading an existing file」でファイルをアップロードすることも可能です（コマンドライン不要）。
3

Vercelでプロジェクトをインポート

  1. vercel.com にGitHubアカウントでログイン
  2. 「Add New Project」→「Import Git Repository」
  3. gc-hr-tools を選択
  4. Framework Preset: Other
  5. Root Directory: そのまま（./）
  6. 「Deploy」をクリック
✅ デプロイ完了後、https://gc-hr-tools-xxxx.vercel.app のようなURLが発行されます。

🅱️ 方法B: Vercel CLIデプロイ

4

Vercel CLIのインストール

npm install -g vercel

デプロイ実行

cd /path/to/gc_package
vercel login    # Vercelアカウントでログイン
vercel          # デプロイ開始
# 質問に答える:
# Set up and deploy: Y
# Which scope: (あなたのアカウントを選択)
# Project name: gc-hr-tools
# Directory: ./
# Override settings: N
vercel --prod   # 本番URLでデプロイ

🅲️ 方法C: ドラッグ&ドロップ

5

最も手軽な方法

  1. vercel.com/new を開く
  2. 「Deploy」セクションにある 「Browse」またはドラッグエリア に gc_package フォルダをドラッグ
  3. プロジェクト名を入力して「Deploy」

⚙️ Vercel設定（vercel.json）

パッケージに含まれる vercel.json には以下の設定が入っています：

{
  "rewrites": [
    { "source": "/", "destination": "/index.html" }
  ],
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        { "key": "X-Frame-Options", "value": "SAMEORIGIN" },
        { "key": "X-Content-Type-Options", "value": "nosniff" }
      ]
    }
  ]
}
